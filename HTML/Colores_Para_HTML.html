Colores Para HTML

Hay muchas formas de ponerle color a una página web con el lenguaje HTML, una de ellas es poner el siguiente atributo a un elemento: style="color: green" y el fondo se hará de color verde, aunque no se recomienda. La mejor solución es usar hojas de estilo. Otro caso sería añadir estilos a la letra, veamos un ejemplo: Texto de prueba y así la letra se pondrá de color verde.
